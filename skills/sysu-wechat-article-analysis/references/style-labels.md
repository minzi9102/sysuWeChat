# Style Label Vocabulary

Use style labels for retrieval by writing mode, emotional posture, and narrative mechanism. Use `article_types` for subject matter.

Every text article starts with:

1. `事实驱动`
2. `分章节叙事`
3. `校媒报道`

Add 2-4 labels from the vocabulary below. New labels are allowed only when no canonical label is equivalent.

## Extension Rules

- Prefer a canonical label over a synonym: use `科研硬核型`, not `硬核科研型`.
- Name a reusable style distinction, not a one-article topic or proper noun.
- End fine labels with `型`.
- Do not repeat a label or copy an `article_types` value verbatim.
- When adding a label, place it in the closest vocabulary section before using it in artifacts.
